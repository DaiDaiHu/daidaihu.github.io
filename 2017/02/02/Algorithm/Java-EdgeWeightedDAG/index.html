[Java]Edge-Weighted DAG(Resizing Image)

Posted on 2017-02-02 | In Application of Algorithm | | Visitors

This is the second assignment of Algorithms II-Princeton-Coursera
This program is to use the idea of edge-weighted DAG to calculate and remove the path of a picture from up to down or left to right that has the smallest energy.
This is used in image resize and ensure keeping the best details.

New for me:

  1. Reducing the unneccessary private function can reduce the using of memory(global variable), although the code will look more complicated.
  2. System.arraycopy can copy and cover it to itself, very interesting.
  3. Defensive copy can also be used in some return function to make sure client cannot mutate it.
